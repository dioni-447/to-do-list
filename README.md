# To-Do List
Aplicação simples e intuitiva para auxiliar na organização de tarefas.

## Funcionalidade:

a propósito o projeto to-do-list é um objeto de estudo para que eu possa desenvolver minhas capacidades e assimilar tudo que venho aprendendo dentro dos meus curso. Como base, o projeto terá funcionalidades como:
-criar tarefas;
-ver tarefas criadas pelo usuário;
-deletar tarefas;
-editar tarefas;

tudo isso será feito com um login que será requerido ao entrar na aplicação.

##como executar:

atualmente o projeto ainda está bem cru, somente com entrada e saída via terminal,
logo, para executá-lo, será necessário ter o python instalado na versão 3 pelo menos.
Após isso, só executar o comando python tarefa.py, para ativar o projeto.

## tecnologias:

foi utilizado ferramentas como, VS code, ambiente na qual uso para desenvolvimento de software.
Além disso, estou usando o GitHub para a publicação dos avanços do projeto.

## o que eu quero alcançar com esse projeto:

o projeto de lista de tarefa, é essencialmente estudantil, sem interesses de lançar mundialmente e etc.
porém conforme eu vá ampliando suas funções, talvez eu decida transformar em um aplicativo mesmo.

Em tese, tenho algumas funções em mente, como por exemplo, adaptar um calendário para que o usuário possa acompanhar suas tarefas com um calendário;
outra função seria agendamento de tarefas com sequências personalizadas, do tipo: "semana sim, semana não tarefa tal se repetirá";
e por fim, uma maneira de conectar o aplicativo com o relógio do celular e agendar despertadores, um dia antes do prazo da tarefa.

São alguns planos que tenho com esse projeto.
